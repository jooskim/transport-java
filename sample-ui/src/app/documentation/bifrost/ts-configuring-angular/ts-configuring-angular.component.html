Configuring Angular

Angular provides angular.json (for Angular 7+) and angular-cli.json (for Angular 4-5).

In this file, there is a section defined as scripts. This section is where you provide third party scripts to be booted and loaded by your Angular application. Clarity already adds a couple of files to this section in order to operate correctly. You will need to add the Bifr√∂st UMD file to this section. It's already been saved in your node_modules folder. Just provide an entry like the one below.

Configuring angular.json

...
"scripts": [
    "node_modules/@vmw/bifrost/bifrost.umd.min.js"
]
...

Your application is ready to go, now you just need to boot the bus.

Angular provides a src/main.ts file, which is essentially your initialization script, that you can use to Initialize the Bus.
