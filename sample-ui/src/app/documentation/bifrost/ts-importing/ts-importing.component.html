Importing the Bifröst into your application

Configuring npm to use VMware's internal npm registry.

Installing the UI / Front-end component is simple, it's published on our internal NPM repository, that is located at https://build-artifactory.eng.vmware.com/artifactory/api/npm/npm.

If you haven't pulled any of VMware's internal projects before, you will need to either configure your ~/.npmrc file to set your registry. You can just add the line registry=https://build-artifactory.eng.vmware.com/artifactory/api/npm/npm to the config. Each time you run npm, it will read you config file.

You can use the command npm config set registry https://build-artifactory.eng.vmware.com/artifactory/api/npm/npm to set your registry.

Installing the Bifröst via npm 

npm install @vmw/bifrost --save
