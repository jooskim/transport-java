Importing the Bifröst into your application

Configuring npm to use VMware's internal npm registry.

Installing the UI / Front-end component is simple, it's published on our internal NPM repository.

If you haven't pulled any of VMware's internal projects before, you will need to either configure your ~/.npmrc file to set your registry, or you can set the registry when you install the module.

The easiest method is just to use our internal artifactory for everything, as it is also a mirror/proxy. You can do this by adding the following line to your ~/.npmrc file. 

registry=https://build-artifactory.eng.vmware.com/artifactory/api/npm/npm

Each time you run npm, it will read your config file, no need to remember moving forward.

Setting your registry using npm

npm config set registry https://build-artifactory.eng.vmware.com/artifactory/api/npm/npm

Installing the Bifröst via npm 

npm install @vmw/bifrost --save
