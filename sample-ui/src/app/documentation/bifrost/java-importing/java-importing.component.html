Importing into your Spring Application (Java)

The Java version of the Bifröst is published as a jar, via our internal artifactory. It can be consumed via gradle, or maven.

Using SS-Base or SS-backend?

If your application is built ontop of ss-base or ss-backend (VMC & Atlas foundation), Then there is nothing to import, you're already set up and ready to go. The Bifröst is already baked in.

What does the Bifröst depend on?

Magic doesn't come for free, so the Bifröst heavily depends on RxJava, Spring, Jackson, Apache Commons and a few others.

Configuring Gradle

The first step is to ensure you add our internal artifactory repository to the list of repositories. 

repositories {
    maven { url "http://build-squid.eng.vmware.com/nexus/content/groups/repo" }
    maven { url "http://build-artifactory.eng.vmware.com/artifactory/bifrost-maven-local" }
}

Then add a dependency to the desired Bifröst jar version and its dependencies. 

dependencies {
    compile "com.vmware.bifrost:vmw-bifrost:0.0.10"
    compile "org.springframework.boot:spring-boot-starter-web:1.5.18.RELEASE"
    compile "org.springframework.boot:spring-boot-starter-websocket:1.5.18.RELEASE"
    compile "org.springframework.security:spring-security-web:4.2.2.RELEASE"
    compile "org.springframework.security:spring-security-config:4.2.2.RELEASE"
    compile "io.reactivex.rxjava2:rxjava:2.1.9"
    compile "com.fasterxml.jackson.core:jackson-databind:2.9.4"
    compile "com.fasterxml.jackson.module:jackson-module-jsonSchema:2.9.4"
    compile "org.slf4j:slf4j-api:1.7.25"
    compile "ch.qos.logback:logback-classic:1.2.3"
    compile "com.google.code.gson:gson:2.8.1"
    compile "joda-time:joda-time:2.9.9"
    compile "org.apache.httpcomponents:httpclient:4.5.6"
    compile "org.apache.commons:commons-lang3:3.8.1"
}

Configuring Maven

Similar to the gradle configuration, first you should add our internal artifactory repository to the list of repositories. 


<repositories>
    <repository>
        <id>Build-Squid-Repo</id>
        <url>http://build-squid.eng.vmware.com/nexus/content/groups/repo</url>
    </repository>
    <repository>
        <id>Bifrost-Repo</id>
        <url>https://build-artifactory.eng.vmware.com/artifactory/bifrost-maven-local/</url>
    </repository>
</repositories>

Next add a dependency to the Bifröst jar and its dependencies.
